「子どもが夜なかなか寝ない…」  
「睡眠時間がバラバラで生活リズムが整わない」

そんなお悩み、我が家も通ってきた道です。

{{< box type="point" title="💡 この記事の結論" >}}
- **朝は毎日同じ時間に起こす**（これだけで夜の眠気が整ってくる）
- **夕食→お風呂→就寝の流れを毎日一定に**する
- **朝に日光を浴びる**
- 1歳と小学生では昼寝の有無など**年齢別に対応を変える**
{{< /box >}}

我が家は**夜9時就寝・朝6時起床**を基本にしています。元保育士として培ってきた知識と、3人の子育て経験をもとに実践していることをお伝えします！

---

## 我が家の基本リズム

| | 時間 |
|---|---|
| 起床 | 6時ごろ |
| 就寝 | 21時（夜9時） |

{{< speech img="/images/profile.png" name="ほいくまま" >}}
最初から完璧を目指さなくて大丈夫！「だいたいこの時間」くらいのゆるさで始めると長続きします🌸
{{< /speech >}}

## 年齢別のポイント

### 👶 1歳の場合：昼寝のスケジュール管理がカギ

1歳はまだ**昼寝が必要な時期**。昼寝の長さ・タイミングが夜の就寝に直結します。

我が家の1歳の1日のリズムはこんな感じです👇

| 時間帯 | 内容 |
|--------|------|
| 起床後 | おもちゃで遊んだりEテレを観たりしてゆったりスタート。ママの家事の間はそれぞれ自由に過ごす |
| 午前中 | 少し活動的に。風船・ボール遊びなど体を動かす遊びや、歩けるようになったら短いお散歩も◎ |
| 午前寝（必要な場合） | **30分程度で短めに**切り上げる。長く寝かせすぎると昼寝・夜の就寝に影響が出る |
| 昼食後〜13時ごろ | お昼寝スタート |
| 13時〜16時の間 | **2時間〜2時間半**お昼寝。16時以降に寝かせると夜の就寝が遅くなりがち |
| 夕方 | 疲れからか機嫌が悪くなりやすい時間帯。夕飯作り中におんぶで**20分ほど**ウトウトさせることも |

{{< box type="note" title="🌿 午前寝のポイント" >}}
1歳でもまだ午前中に眠くなる子はいます。午前寝をする場合は**30分程度で短めに**切り上げるのがポイント。長く寝かせすぎると、肝心の昼寝が取れなくなったり、夜の就寝が遅くなる原因になります。
{{< /box >}}

{{< speech img="/images/profile.png" name="ほいくまま" >}}
夕方のグズグズはうちの末っ子あるある😅 夕飯を作りながらおんぶで少し寝かせると、ご機嫌が戻って夜のお風呂までスムーズに進めます！
{{< /speech >}}

### 🎒 小学生の場合：昼寝なし・朝のリズムが大事

小学生になると昼寝はなくなり、**朝のリズムが一日の調子を左右**します。

{{< box type="good" title="✅ 小学生でやっていること" >}}
- **夜9時には布団に入る**ことを習慣に（「9時ルール」として家族で決めています）
- 朝は毎日同じ時間に起こす（休日も大きくズレないように）
- 帰宅後は**宿題→夕食→お風呂**の流れを固定する
{{< /box >}}

{{< speech img="/images/profile.png" name="ほいくまま" >}}
小学生は「眠くなったら寝る」が自分ではなかなかできません。親が「9時になったよ」と声をかけるのが大事。うちは「9時ルール」と呼んでいて、子どもたち自身も自然と守れるようになりました😊
{{< /speech >}}

## リズムを整えるうえで大事にしている2つのこと

### ☀️ 朝、日光を浴びる

朝に日光を浴びると体内時計がリセットされ、夜になると自然な眠気が生まれます。

カーテンを開けて朝日を取り込むだけでOK。1歳の子を連れて短いお散歩ができると、**活動量も上がって昼寝がスムーズに**入りやすくなります。

### 🛁 夕食→お風呂→絵本→就寝の流れを毎日一定にする

毎日同じ順番で寝る準備をすることで、**「この流れになったら寝る時間」と子どもの体が覚えていきます**。

我が家では**寝る前に必ず絵本タイム**を設けています。布団に入る前に絵本を読むことをルーティンにすることで、子どもが「絵本が終わったら寝る時間」と自然に理解できるようになります。

{{< box type="good" title="📚 絵本タイムは良いことづくし！" >}}
- **言葉の発達**につながる
- 親子のスキンシップを通した**愛着関係の形成**に効果的
- 静かな時間を作ることで**自然と眠気**が生まれる
- 読み聞かせの声が子どもをリラックスさせる
{{< /box >}}

{{< speech img="/images/profile.png" name="ほいくまま" >}}
保育園でも「給食→昼寝」の流れを毎日繰り返すうちに、子どもたちが自然に眠くなっていきます。ルーティンの力って本当にすごい。絵本タイムを加えてからは、子どもが自分から「絵本読んで」と持ってくるようになりました。その流れでスムーズに就寝できています✨
{{< /speech >}}

## リズムが崩れたときはどうする？

キャンプや旅行、体調不良でリズムが崩れることはよくあります。

{{< box type="note" title="🌿 崩れてもあわてない！" >}}
- **翌朝は同じ時間に起こす**（就寝が遅くても起床時間は変えない）
- 1〜2日続ければ元に戻ることがほとんど
- 「完璧に守らなきゃ」と思いすぎない
{{< /box >}}

{{< speech img="/images/profile.png" name="ほいくまま" >}}
我が家はキャンプのたびにリズムが崩れます（笑）でも「翌朝は同じ時間に起こす」だけ意識すれば、2〜3日で戻ります。崩れること自体は気にしすぎなくて大丈夫です！
{{< /speech >}}

---

## まとめ：まず「起きる時間」を固定するだけでOK

子どもが夜寝ない・睡眠時間が整わないときは、就寝時間を早めようとするより、**まず朝起きる時間を固定する**のが近道です。

{{< box type="good" title="✅ 今日からできること" >}}
1. 毎朝同じ時間にカーテンを開けて日光を浴びる
2. 夕食→お風呂→**絵本タイム**→就寝の順番を決めて毎日繰り返す
3. 【1歳】午前寝は30分程度に抑え、昼寝は13〜16時の間に2時間程度
4. 【小学生】「何時になったら布団に入る」ルールを作る
{{< /box >}}

{{< speech img="/images/profile.png" name="ほいくまま" >}}
リズムが整うと子どもの機嫌が良くなって、ママにも夜の自由な時間が生まれます。完璧じゃなくていいので、まず「朝の起こす時間」だけ決めるところから始めてみてください🌸
{{< /speech >}}
